# 跨平台音乐播放器

一个现代化的Web音乐播放器，支持多平台音乐源搜索和播放，具有美观的用户界面和丰富的功能。

##  功能特性

### 核心功能
- **多平台音乐源支持**：Demo源、JK无铭API、多源（普通+无铭）
- **音乐搜索**：支持关键词搜索歌曲
- **播放控制**：播放/暂停、上一首/下一首、随机播放、单曲循环
- **进度控制**：拖拽进度条、音量调节
- **歌词显示**：实时歌词同步显示
- **歌单管理**：添加/移除歌曲、拖拽排序、保存/清空歌单

### 用户体验
- **响应式设计**：适配桌面和移动设备
- **快捷键支持**：Space播放/暂停，←/→跳转，↑/↓音量
- **本地存储**：歌单自动保存到浏览器本地存储
- **现代化UI**：深色主题，渐变背景，毛玻璃效果

## 快速开始

### 环境要求（请使用git bash）
确保你已经安装了Node.js和npm（Node包管理器）。

克隆项目到本地：

cd qq-music-api
bash

启动服务器
执行以下命令启动QQ音乐API服务：

## 使用说明

### 搜索音乐
1. 在顶部搜索框输入歌曲名称
2. 选择音乐源平台
3. 点击搜索按钮
4. 从搜索结果中选择歌曲

### 播放控制
- **播放/暂停**：点击播放按钮或按空格键
- **上一首/下一首**：使用控制按钮
- **随机播放**：点击随机按钮
- **单曲循环**：点击循环按钮
- **进度跳转**：拖拽进度条
- **音量调节**：使用音量滑块

### 歌单管理
- **添加歌曲**：从搜索结果点击"加入歌单"
- **立即播放**：点击"立即播放"直接播放
- **移除歌曲**：在歌单中点击"移除"
- **排序**：拖拽歌单项目重新排序
- **保存歌单**：点击"保存歌单"按钮
- **清空歌单**：点击"清空"按钮

## 技术架构

### 前端技术
- **HTML5**：语义化结构
- **CSS3**：现代样式，Grid布局，CSS变量
- **JavaScript ES6+**：模块化代码，异步处理

### 音乐源适配器
项目采用适配器模式，支持多种音乐源：

#### DemoAdapter
- 提供示例音乐和歌词
- 无需外部API，适合演示

#### JkApiAdapter
- 集成JK无铭API
- 需要API密钥配置
- 支持网易云音乐源

#### CommonAdapter
- 通用音乐适配器
- 支持QQ音乐源
- 需要本地适配器服务

#### MultiCommonJkAdapter
- 多源合并适配器
- 结合普通适配器和JK API
- 智能去重和字段合并

### 核心功能模块

#### 播放器引擎
- HTML5 Audio API
- 播放状态管理
- 进度同步
- 错误处理

#### 歌词系统
- LRC格式解析
- 时间戳同步
- 实时高亮显示
- 自动滚动

#### 数据持久化
- LocalStorage存储
- 歌单状态保存
